- responsive tailwind (sm:...)
    - focus ใช้สำหรับการกำหนดสไตล์เมื่อองค์ประกอบได้รับการโฟกัส (focus) เช่น เมื่อผู้ใช้คลิกหรือใช้แป้นพิมพ์เพื่อเลือก
    - active ใช้สำหรับการกำหนดสไตล์เมื่อองค์ประกอบถูกกด (active) เช่น เมื่อผู้ใช้คลิกและกดค้างที่องค์ประกอบนั้น
